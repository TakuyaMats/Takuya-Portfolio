Product Feedback App
This is a product feedback web application that I have built that allows users to provide feedback and suggestions to improve the application, as well as share ideas related to their current work. The website is a full stack application that enables users to create an account and contribute to the forum. The design for the website was inspired by a Figma file provided by Frontend Mentor. As the sole developer, I designed and built the back-end API, showcasing my proficiency in back-end development and API design. TECHNOLOGY
HTML | SASS | Javascript | handlebars.js | mysql | sequelize

Space Tourism
As part of a collaborative effort with two other developers, I contributed to the development of a mock-up website named 'space-tourism.' The purpose of the website is to provide users with a comprehensive understanding of what to expect when planning a trip to space. Our team built the website on the React framework, which provided us with an excellent opportunity to expand our technical capabilities and improve our coding skills. During the project, my primary responsibility was to mentor my teammates in common git commands and git flow. I taught them the importance of using git branches when implementing new features, which enabled us to collaborate more efficiently and streamline our development workflow. TECHNOLOGY
HTML | CSS | React | Material-UI | React Router Dom
